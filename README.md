# 组件化和插件化的区别 ： 
* 相同点 ： 将一整个项目，分而治之，降低耦合
* 不同点 ： 
  * 组件化 ： 编译期
  * 插件化 ： 编译期，运行期

# 组件化要点 ： 
* 代码解耦
* 组件单独运行
* 数据传递
* UI跳转
* 组件的生命周期
* 集成调试
* 代码隔离
